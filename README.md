# GigShield AI

GigShield AI is an AI-powered parametric insurance prototype for gig workers such as delivery partners.

## Problem

Gig workers lose income due to heavy rain, heatwaves, app outages, and other external disruptions.

## Solution

GigShield AI automatically detects such disruptions and simulates instant claim payout using predefined parametric triggers.

## Features

- Worker login and dashboard
- Risk level indicator
- Trigger simulation for rain, heat, and app failure
- Live weather check
- Claim generation
- Payment success simulation
- Admin dashboard

## Tech Stack

- React Native
- Expo
- JavaScript / TypeScript
- Open-Meteo API
